:hover

Example: Hover over a link.
Hover over me 
                        
                            a {
                                text-decoration: none;
                                transition: 0.3s;
                            }
                            a:hover {
                                text-decoration: underline;
                                font-weight: bold;
                                font-size: 2em;
                                color: orange;
                            }

:focus

Example #2: User focus is on Input field. Click inside the input field. 
                        
                            #focus .interactive {
                                display: flex;
                                flex-direction: row-reverse;
                                align-items: center;
                                justify-content: flex-end;
                            }
                            
                            #focus .interactive input,
                            #focus .interactive label {
                                transition: 0.3s;
                                margin: 0 1ch;
                            }
                            
                            #focus .interactive input:focus {
                                border: 4px solid purple;
                                padding: 0.5em 1em;
                                border-radius: 0.5em;
                            }
                            #focus .interactive input:focus + label {
                                font-style: italic;
                                text-decoration: underline;
                                color: purple;
                            }

:checked

Example #3: Input field is a checkbox type and is selected. 
                        
                            #checked .interactive input:checked {
                                outline: 2px solid darkgreen;
                            }

                            #checked .interactive input:checked + label {
                                font-weight: bold;
                                text-decoration: underline;
                            }

:required

Example: Input field is required. 
                        
                            input:required {
                                border: 3px dotted blue;
                            }
                            input:required:focus {
                                border: 3px dotted green;
                                background-color: darkgoldenrod;
                            }
                            
                            /* Add an asterix with an After element */
                            #required .interactive input:required + label::after {
                                content: "*";
                                color: red;
                                font-weight: bold;
                                margin-right: 1em;
                            }
                            #required .interactive {
                                display: flex;
                                flex-direction: row-reverse;
                                justify-content: flex-end;
                            }
